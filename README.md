# КуРус - идеальный самолёт-тренер до 150г!

[![CC BY-NC 4.0][cc-by-nc-shield]][cc-by-nc]
![alt text](https://github.com/YuRa-Aero/Ku-6-FPV-Plane/blob/images/Ku-6.jpg?raw=true)

Самолет подходит для полётов в зале и на улице. Он легко собиарется из 5 частей и имеет возможность установки FPV и полетного контроллера от вупов

 
___
**Некоторые интересные особенности "КуРуса"**

- Применена уникальная технология создания петель. Они вырезаются сразу на лазере и управляющие поверхности не требуют последующей навески. Отклонение происходит за счет повышенно гибкости в местах с вырезанным паттерном
- Фюзеляж собирается из 3х частей. Внутри сразу вырезаны все посадочные отверстия и ложементы для проводов
- Специальная конструкция крыла позволяет найти компромисс между прочностью, созданием подъемной силы и весом. Конструкция представляет из себя изогнутый лист, который принимает форму профиля. Такая технология позволяет значительно сэкономить вес

___

**Электроника** :
- 12А ESC
- 3x 3,7G сервопривод
- 2s 450mah LiPo аккумулятор
- Мотор 3750KV
